# Mapsage

* This is a MEVN web application
* Mapsage is a social network for masseurs and their costumers
* Backend server based on node/express
* Frontend based on vue-cli

# Installation

* Clone or download repository

```
git clone https://github.com/nem3s1s/mapsage-for-good.git
```

# Set up:

### Frontend Set Up
```
cd mapsage-for-good
cd frontend
npm install
```

### Backend Set Up
```
cd mapsage-for-good
cd backend
npm install
```

# Run Application:


### Run Backend server:
```
cd mapsage-for-good
cd backend
node index.js
```

### Run Frontend:
```
cd mapsage-for-good
cd frontend
npm run serve
```
